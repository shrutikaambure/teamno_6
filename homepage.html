About the Course

The Design and Analysis of Algorithms refers to the study of designing algorithms that efficiently solve computational problems while analyzing their performance in terms of time complexity and space complexity. It involves techniques such as sorting, backtracking, recursive algorithms, and graph-based solutions.

Data Structures and Algorithms Covered

Data Structures & Algorithms Description
Fundamentals of Algorithm and Problem Solving Space & Time Complexity, Recursion, Backtracking, Stacks, Queues.
Hashing and Hash Tables Hash Tables, Linear Probing, Quadratic Probing, Bucket Table, Sparse Table.
Graphs and Trees Graphs, AVL Tree, 2-3 Trees, Binary Tree, DFS, BFS, Tries.
Sorting Algorithms Bubble Sort, Selection Sort, Merge Sort, Quick Sort, Heap Sort.
Substring Search Algorithms Brute Force Algorithm, Boyer-Moore Algorithm, KMP Algorithm, Rabin-Karp Algorithm.
Graph Algorithms Floyd Algorithm, Warshall's Algorithm, Dijkstra's Algorithm, Bellman Ford Algorithm,Prims Algorithm,Kruskal Algorithm
Problem Cases Traveling Salesman Problem, Knapscak problem,Fake Coin Problem, Huffman Code, Strassen's Matrix Multiplication.
